Official Integrations

Official integrations are pre-built, CatalEx-maintained connections to popular services. They give your agents a fast, reliable way to work with the tools your team already uses — no custom development required. CatalEx handles updates and compatibility for you.

Connect and manage all of these from the Tools page. Once connected, an integration's actions become available to your Studio agents and to Freeflow.

info

Connecting integrations requires the ADMIN or OWNER role. Some integrations connect with one click; others need your own API key or OAuth app (marked Setup Required on the Tools page). A few may show as Coming Soon. The Tools page always reflects current availability.

Built-in tools (always available)​

These work out of the box with no setup:

  • Web Search — search the web for up-to-date information.
  • Web Fetcher — fetch and extract content from web URLs.
  • Script Executor — run sandboxed Python or shell scripts.
  • Knowledge Base — search your company's synced documents and conversations.
